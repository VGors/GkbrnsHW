1. Драйвер для работы с SQLight добавлен через зависимости в Maven.
2. База данных пользователей сгенерирована заранее через пакет Batch. В ней присутствует доп. столбец, который не
понадобился, но может в дальнейшем пригодится. Ключем является ник пользователя.
3. Соединение с БД открывается при старте сервера и закрывается при завершении работы сервера.
4. Ники меняются, учитывается уникальность.
