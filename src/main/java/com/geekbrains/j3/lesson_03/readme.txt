1. Созданы интерфейсы для логирования и цензуры.
2. При успешном подключении клиента проверяется существует ли файл лога для этого ника или не является ли указанный путь
папкой, если да, создаётся новый файл лога (для каждого ника свой файл).
3. При вызове метода можно выбрать кол-во подгружаемых строк из файла лога.
4. При запуске сервера заполняется карта цензуры из файла.
5. Во всех пунктах реализованы проверки есть ли файл и не является ли он папкой.
6. В конце работы ридеры и райтеры закрываются.
