   Что реализовано:
1. При инициализации, каждый участник получает суммарные высоту препятствий и длину дистанции, которые он может преодолеть.
2. При прохождении каждого участка из массива препятствий, уменьшается соответствующий параметр у участника.
3. На входе любой из участников получает массив препятствий, каждое с собственной характеристикой (высота, длина).
4. Каждый из участников по очереди пытается пройти дистанцию - ему передаётся массив с препятсвиями.
5. Если какой-то из ресурсов (бег, прыжки) кончается, то участник сходит с дистанции и выводится краткая информация о
   преодолённых участках. В противном случае участник успешно финиширует.

   Какие возможны улучшения:
1. Автоматическое заполнение входных массивов препятсвий и участников (стандартный набор имён хранится в enum).
2. Возможна реализация определения победителя - добавить скорость прохождения.
3. Добавление случайно сгенерированной помехи в виде падения.
